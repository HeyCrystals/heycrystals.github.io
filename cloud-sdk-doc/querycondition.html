条件查询

目前条件查询支持四种匹配模式，fuzzy(模糊)、term(匹配)、rang(范围)、prefix(前缀)

三种大前提：

  • must：必须满足
  • must_not：必须不满足
  • should:应该满足

查询结构体如下：

{
    "page_size": 10,
    "current_page": 1,
    "token": "JDUkV1ZuRnBUV1AkYVk0Z2R3RVJyem1udktWS1FTQjd1amdSMUlRUXN3V2JrVkl3M3BWTzdQLw",
    "display_fields": "",
    "query": {
        "must_not": [],
        "should": [],
        "must": [
            {
                "prefix": {},
                "term": {
                    "member_id": "f09db39e40ef44a18d4b61f14fbecc37"
                },
                "fuzzy": {},
                "range": {}
            }
        ]
    },
    "sort": {},
    "rel": []
}

代码示例：

CloudBasicConnection cloudBasicConnection=new CloudBasicConnection(); 
cloudBasicConnection.setUrl("y.drore.com"); 
cloudBasicConnection.setPort(80); 
//appid & appsecret
cloudBasicConnection.setUserName("xx"); 
cloudBasicConnection.setPassword("test"); 
//创建连接池管理类 
LogbackLogger.info("create cloudPoolingConnectionManager..."); 
CloudPoolingConnectionManager cloudPoolingConnectionManager=new CloudPoolingConnectionManager();
cloudPoolingConnectionManager.setConnection(cloudBasicConnection); 
//创建数据源 
LogbackLogger.info("create cloudBasicDataSource..."); 
CloudBasicDataSource cloudBasicDataSource=new CloudBasicDataSource(); 
cloudBasicDataSource.setCloudPoolingConnectionManager(cloudPoolingConnectionManager); 
//创建查询runner 
LogbackLogger.info("create cloudQueryRunner..."); 
CloudQueryRunner runner=new CloudQueryRunner(); 
runner.setDataSource(cloudBasicDataSource);

以上是CloudQueryRunner创建代码,一个项目中只需要创建一次,使用spring依赖注入即可

以下代码是条件查询示例：

//创建查询条件
JsonObject jsonObject=new JsonObject();
RequestExample requestExample=new RequestExample(10,1); 
//创建Criteria 
RequestExample.Criteria rc = requestExample.create();
RequestExample.Param pa = requestExample.createParam();
if(jsonObject.has("memberId")){
     pa.addTerm("member_id",jsonObject.get("memberId").getAsString());
}
if(jsonObject.has("orderNo")){
     pa.addTerm("order_no",jsonObject.get("orderNo").getAsString());
}
rc.getMust().add(pa);
//requestExample.setDisplayFields(""); 
GsonBuilder gsonBuilder=new GsonBuilder(); 
gsonBuilder.setPrettyPrinting(); 
System.out.println(gsonBuilder.create().toJson(runner.queryListByExample("scenic_info",requestExample)));
