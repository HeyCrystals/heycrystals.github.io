数据操作(增加、删除、修改)

数据中心数据操作

数据增加

代码示例：


//...创建CloudQueryRunner

Map record=new HashMap();
record.put("name","张三");
record.put("age",20);
//插入,person表，提供key/value数据结构数据
run.insert("person",record);

List records=new ArrayList();
records.add(record);
//批量插入
run.insertBatch("person",records);

数据修改

代码示例:

//...创建CloudQueryRunner
Map record=new HashMap();
record.put("name","张三1");
record.put("age",22);
//更新,person表，提供key/value数据结构数据，需要提供主键pkid
run.update("person","1d7840949f0847b882b30421610704d5",record);
List records=new ArrayList();
//批量更新需提供主键
record.put("pkid","1d7840949f0847b882b30421610704d5");
records.add(record);
//批量插入
run.updateBatch("person",records);

数据删除

代码示例:

//...创建CloudQueryRunner
//根据主键pkid删除
runner.delete("person","1d7840949f0847b882b30421610704d5");

//根据条件删除,删除所有年龄22岁的数据
Map record=new HashMap();
record.put("age",22);
runner.deleteByCriterion("person",record);
