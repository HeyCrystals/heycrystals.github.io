简介

cloud-sdk

cloud-sdk是云数据中心操作工具包,提供数据库、包、资源等操作工具类,同时还包含第三方工具类(快递鸟、TDP平台)

cloud-sdk 目标

  • 提供简单类似工厂调用方式调用操作云数据中心

  • 提供一些util等工具类

  • 提供spring依赖注入方式,更加方便

  • 统一后端和数据中心对接的通道,方便以后项目维护、代码风格统一
